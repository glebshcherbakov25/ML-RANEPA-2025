
## Брильянты

Ищем стоимость брильянтов...

Все камни считаются [круглой огранки `Round`](https://www.bluenile.com/diamonds/round-cut?srsltid=AfmBOoo5oApWGOk0oAhAox232DVbN_Li3kK0ECwP-jZ3CwbWEJFXOO9h), [нижняя площадка отсутствует `None`](https://www.fionadiamonds.com/pages/culet)и все камни имеют [сертификат GIA](https://lavivion.ru/GIA-certificate/)

Остальные параметры легко загуглить
`size`, 
`color`, 
`fancy_color_dominant_color`,
`fancy_color_secondary_color`, 
`fancy_color_overtone`,
`fancy_color_intensity`, 
`clarity`, 
`cut`, 
`symmetry`, 
`polish`,
`depth_percent`, 
`table_percent`, 
`meas_length`, 
`meas_width`,
`meas_depth`, 
`girdle_min`, 
`girdle_max`, 
`culet_condition`,
`fluor_color`, 
`fluor_intensity`,  
`eye_clean`

Ну и по мелочам датасет подчищен ,,,


Целевая переменная -  `total_sales_price`

Целевая метрика - **Mean squared logarithmic error (MSLE)**
